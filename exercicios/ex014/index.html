Para usar estilos css internos, deve se criar uma tag <style> acima do title, e em seguida criar parâmetros para tags, por exemplo:

body{
    background-color: lightsteelblue;
    font-family: Arial, Helvetica, sans-serif;
    font-size: 20px;
}

Esse código serve para mudar a cor de fundo do body e mudar a fonte e o tamanho de todos os textos que estejam escritos no body. Deve-se usar essa forma para mudar os estilos. Porém há algumas desvantagens nesse tipo de uso de CSS3. A primeira desvantagem é o tamanho de espaço que os styles ocupam, a outra desvantagem é que o códigos dos style não passa de página para página caso eu tanho várias em meu site/na pasta, o que signifique que caso eu queria que todas sigam um padrão, eu necessitaria de copiar o style e colar em cada uma delas, o que é nada prático. Mas para projetos de site simples, atende bem.
