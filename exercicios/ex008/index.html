Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica(não possui significado, apenas forma)).

Nesta frase, tenho um termo em destaque, usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Texto marcado

Podemos criar também um texto marcado, usando a tag <mark>

para mudar a cor do marcador é necessário usar CSS3, em HTML5 não há uma forma de mudar a cor.

Dessa forma <mark style="background-color: cor desejada">

Para que todos os marks mantenham um padrão, é necessário criar uma tag <style> acima de <head></head> para os marks. Dessa forma
<style>

mark
{
background-color: cor desejada;
}
</style>

(meu código está cheio de <br> em excesso e exagero, mas apenas porque ainda não sei fazer o que desejo em CSS3).

Agora demonstrando esse padrão: todos os textos marcados com a tag <mark> estarão com o mesmo padrão de cor.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Para isso, usamos as tags <big> e a tag <small>. No entanto, como a tag <big> não tem sentido semântico já está obsoleta, logo, caso haja a necessidade de aumentar um texto, isso deve ser feito no CSS3. Já a tag <small>, tem sentido semântico e pode ser usada no código HTML5.

Para isso, usa-se a tag <del>.
