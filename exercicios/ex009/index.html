Trabalhando com listas

Há como fazer vários tipos de listas ordenadas, utilizando a tag <ol> para delimitar a lista toda e <li> para delimitar cada item. Também há como aplicar um parâmetro type nas listas ordenadasm há como fazer listas numeradas utilizando o parâmetro type 1 (que já é o padrão automático), listas alfabéticas maiúsculas utilizando parâmetro type A, listas alfabéticas minúsculas utilizando o parâmetro type a, listas em algarismos romanos maiúsculos utilizando o parâmetro type I e listas em algarismos romanos minúsculos utilizando o parâmetro type i. Dentro do parâmetro type, há como selecionar em que letra/número a lista começará, usando o start. Dessa forma: <ol type="1" start="4"> para listas numeradas, por exemplo.

Listas não ordenadas

Há também as listas não ordenadas, também conhecidas como listas com marca, essas são as lista que possuem marcadores atrás, como bolinhas vazias (circle), bolinhas pretas (disc, que já é o padrão automático) e quadrados (square). Para criar uma lista não ordenada utiliza-se a tag <ul>, e para escolher o seu marcador, utiliza-se o parâmetro type. Dessa forma: <ul type="circle"> para fazer uma lista demarcada por bolinhas vazias por exemplo.

Misturando listas

Há como criar sublistas dentro das listas principais, podendo ser ol's ou ul's, para que as listas principais se tornem um "tópico" superior as sublistas que seriam a sua "descrição". Dessa forma: 

<li>Antigas</li> 
<ul type="estilo de marcador desejado">
    <li>C</li> 
    <li>C++</li> 
    <li>PHP</li> 
    <li>Java</li> 

<li>Novas</li> 
<ul type="estilo de marcador desejado">
    <li>Javascript</li> 
    <li>Python</li> 
    <li>C#</li> 
    <li>Ruby</li>
  1. Antigas
    • C
    • C++
    • Delphi
    • Java
  2. Novas
    • Javascript
    • Python
    • C#
    • Ruby

Lista de Definições

As listas de definição são parecidas com dicionários, que há um termo e sequencialmente há a sua definição, e para fazê-la usa-se a tag <dl> para englobar toda a lista, <dt> para englobar o termo a ser definido e <dd> para dar a definição desse termo.
