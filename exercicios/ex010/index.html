Usando links externos

Para usar um link externo (ou âncora) é necessário utilizar a tag <a> e colocar a URL desejada no campo href (referência de hipertexto). Dessa forma: <a href="https://www.youtube.com/@CursoemVideo">

Você pode acessar o canal do Curso em Vídeo

Você pode acessar também o repositório público do Guanabara no GitHub

Para colocar uma URL externa e abri-lá sem ela sobrepor o seu site, usa-se o paramêtro target="_blank" (janela nova em branco) e o rel="external" (link externo ao meu site). Dessa forma: <a href="https://www.netflix.com/browse target="_blank" rel="external">

Usando links internos

Links internos farão uma ligação com outra página que esteja dentro do meu próprio servidor. Para criar um link interno é necessário ter uma outra página. E para fazer isso é necessário implementar um arquivo secundário dentro do index, esse arquivo tem que estar na mesma página do index. Dessa forma:<a href="outrapágina.html" rel="next">

Esta é a primeira página do site, se você quiser pode acessar também a minha segunda página

Caso o arquivo esteja dentro de uma pasta, é necessário informar o seu caminho. Dessa forma: <a href="nomedapasta/pag003.html">página de notícias>
