Para usar estilos CSS externos, é necessário criar um arquivo dentro da pasta do index com o nome que você desejar .css, em seguida você coloca os parâmetros desejados para o seu site nesse arquivo. Para integrá-lo ao HTML5, é necessário ir acima do title e colocar a seguinte tag: (digite link e aparecerá a opção link:css) <link rel="stylesheet" href="nomedoarquivo.css">

Pode-se usar mais de um link .css para integrar mais de um arquivo CSS ao HTML, com alguma/s página/s que tenham configurações específicas que devem ser aplicadas somente ao desejado.

Essa maneira é a mais versátil de se usar e integrar CSS3 ao HTML5.
